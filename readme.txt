This plugin add widget which displays random links on your website.

== Description ==
Simple widget based on the default Links widget. It displays random links on your website. 

**I am open to suggestions to improve the plugin !**

== Installation ==
Activate the plugin and add "Rotating Links Widget" to your dynamic sidebar.

You can set the title, the number of links to show, the links category and some other options in the widget options.
